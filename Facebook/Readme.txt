Start the Xampp control panel and transfer all your files to the 'htdocs' directory. Then, initiate both the Apache web server and the MySQL database. Once in MySQL, establish a new database called 'raphntc'. Finally, import the database file, also named 'raphntc', which you'll find within the MySQL folder.

Inicie o painel de controle do Xampp e transfira todos os seus arquivos para o diretório 'htdocs'. Então, inicie o servidor web Apache e o banco de dados MySQL. Uma vez no MySQL, estabeleça um novo banco de dados chamado 'raphntc'. Finalmente, importe o arquivo do banco de dados, também chamado 'raphntc', que você encontrará dentro da pasta MySQL.
